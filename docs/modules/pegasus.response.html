Module pegasus.response

Module pegasus.response

HTTP response writer used by your application callback.

Instances are created internally by Pegasus and passed to server:start(function(request, response) ... end).

Quick example: ```lua server:start(function(req, res) res:statusCode(200)

:contentType('application/json')
:write('{"ok":true}')

end) ```
